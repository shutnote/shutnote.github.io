Name: Speedy Delievery

Description: Speedy Delievery was a mobile game briefly published on the android app store. It was a university group project where I was the programmer for the project. Overall the project achieved a 62% mark.

Features:

This cutscene system is completely custom

Nodes are placed, these nodes can lead to other nodes for camera movemment, prop movement(Such as NPC, etc) and call functions when triggered. There are also zones that wait till a condition is meet. Used in the tutorial section
