Shooter Multiplayer Game

This is a unity game made using unity netcode.

The game is heavily inspired by Halo

It features player customisation, multiple weapons, map voting, mode voting

Features:

The game features 5 different weapons, each with there own customisable properties

Nodes are placed, these nodes can lead to other nodes for camera movemment, prop movement(Such as NPC, etc) and call functions when triggered. There are also zones that wait till a condition is meet. Used in the tutorial section

The game features multiplayer!
