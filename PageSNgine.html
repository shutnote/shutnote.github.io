SNgine

SNgine is a 3D game engine built using C++, opengl and the Bullet physics engine.

The project included Window, keyboard, mouse and controller inputs, physics along with three types of rigidbody(Cube, Sphere, Capsuel), terrain, objects, cells/world space, Camera, UI rendering, Text rendering, UI buttons, Save/Loadings.

After the submission of the project, development continued by overhauling many of the systems present. This included the camera system, asset management, UI + Text rendering and images.

Overall this project achieved a 79%

Class Name: Window

Namespace: SN

Usage: To create and manage the main window that the engine uses for drawing, inputs, etc.

Functions:

This is the constructor class, it takes no parameters. Its used to start up the openGL context, set the resultion of the window, give it a name and set the openGL settings.

It also loads the default shaders into a main shader program as well as the Post processing and UI shader. Creates a basic quad that the camera is rendered onto.

It also starts up DS5W which is used for dualsense controller inputs(thought this feature is legacy).

This is the Inital Update function, it takes no parameters. It's called at the start of every frame.

It gets the window up time for the frame time/delta time calculation. It also clears the windows buffers(Colour and Depth).

It also handles the mouse escape and VSync inputs before looping through each of the Inputs created by a user.
