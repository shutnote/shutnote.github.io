Combat Chaos

Combat Chaos is a multiplayer game build in the unity game engine using the new Unity Netcode module.

I was the programmer in charge of the gamemodes(deathmatch, CTF, King of the hill, infection), the logic behind them and the selection logic.

I created a gamemode framework which would allow easy tweaks of gamemode data such as score, starting weapons, team sizes, etc.

I also made the shooting logic.

The game features four main game modes, each of which had a spin off. Deathmatch and team snipers, capture the flag and ???, king of the hill and ???, infection and hide and seek.

Overall this project achieved a 85%
